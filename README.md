# Unsupervised Learning

Clustering and dimensionality reduction techniques applied to real business datasets — customer segmentation, tourism service grouping, and product portfolio analysis.

## Notebooks

| Notebook | Dataset | Technique |
|---|---|---|
| `USL_Project_LearnerNotebook_FullCode-1.ipynb` | Full project | K-Means + Hierarchical |
| `USL_MLS1_EngineeringCollege_v1_1.ipynb` | Engineering college data | Clustering analysis |
| `USL_MLS2_TourismServices (1) (1).ipynb` | Tourism services | Customer segmentation |
| `USL_W2_AdditionalCaseStudy_ProductSegmentation.ipynb` | Product data | Market segmentation |

## Techniques Covered

- **K-Means Clustering** — elbow method for optimal k, centroid analysis
- **Hierarchical / Agglomerative Clustering** — dendrograms, linkage methods
- **PCA (Principal Component Analysis)** — variance explained, scree plots, biplot interpretation
- **Cluster Profiling** — business interpretation of each cluster
- **Silhouette Score** — quantitative cluster quality evaluation

## Business Applications

- Customer segmentation for targeted marketing
- Product portfolio grouping for pricing strategy
- Service bundle recommendation based on usage patterns

## Tech Stack

- **Libraries:** scikit-learn, SciPy, Pandas, NumPy, Matplotlib, Seaborn
- **Runtime:** Python, Jupyter Notebook

## Skills Demonstrated

K-Means, hierarchical clustering, PCA, dimensionality reduction, silhouette analysis, cluster interpretation, business-driven segmentation
